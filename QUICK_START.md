# ⚡ FanVault 快速入门指南

## 🎯 5分钟快速开始

### 步骤 1: 安装依赖

```bash
pnpm install
```

### 步骤 2: 配置环境变量（可选）

Stripe密钥已经预配置在代码中，你可以直接使用。如果需要使用自己的密钥：

```bash
cp .env.example .env
# 然后编辑 .env 文件，填入你的 Stripe API 密钥
```

### 步骤 3: 启动应用

```bash
pnpm dev
```

访问 `http://localhost:5173` 🎉

## ✨ 核心功能快速体验

### 1️⃣ 创建内容 (Creator功能)

1. 点击右下角的 **蓝色加号按钮**
2. 上传图片 → **自动打开图片编辑器**
3. 编辑图片（裁剪、旋转、调整亮度等）
4. 上传视频 → **自动打开视频处理器**
5. 点击 "Set Schedule" → **打开定时发布模态框**
6. 发布内容！

### 2️⃣ 数据分析

1. 侧边栏 → **Analytics**
2. 查看收入趋势、订阅者增长、内容表现等
3. 使用过滤器查看不同时间段的数据

### 3️⃣ 实时通知

1. 顶部导航栏 → **铃铛图标**
2. 查看实时通知（打赏、订阅、评论等）
3. 点击通知快速跳转

### 4️⃣ Stripe 支付测试

1. 点击任意创作者的 **Subscribe** 按钮
2. 使用测试卡号: `4242 4242 4242 4242`
3. 过期日期: 任意未来日期（如 `12/34`）
4. CVC: 任意3位数（如 `123`）
5. 完成支付！

## 🐛 常见问题速查

**Q: 支付测试失败？**
→ 使用测试卡号 `4242 4242 4242 4242`

**Q: 图片编辑器不显示？**
→ 确保上传的是图片格式（JPG/PNG/WebP）

**Q: 视频处理很慢？**
→ 正常，客户端处理需要时间，可以选择"Low"质量加快速度

**Q: 通知不更新？**
→ 当前是模拟数据，每30秒自动添加新通知

**Q: 找不到某个页面？**
→ 确保以创作者身份登录（自动启用）
